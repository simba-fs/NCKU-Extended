# NCKU-Extended
成大系統瀏覽器擴充套件

擴充成大選課系統的功能  
目前功能還不多，有其他想法歡迎發 Issue

## 功能
1. 記住[選課頁面](https://course.ncku.edu.tw/index.php?c=qry11215&m=en_query)篩選條件
2. 幫當[學期成績查詢](http://eportfolio.ncku.edu.tw/index2/stu/index.php?t=ss)加上分佈曲線的連結
